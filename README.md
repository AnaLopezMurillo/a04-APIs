# a04-APIs
A04 Assignment - Public APIs

The website logs real-time as well as upcoming weather for the UNC Chapel-Hill areas in a 90s style display. 

Weather icons are made by logging WMO Weather intepretation codes and layered based on intensity.

All icons and graphics were hand-drawn using Adobe Illustrator.

## APIs used:
https://open-meteo.com/
https://www.7timer.info/doc.php?lang=en
    takes in lat, long, product "astro", "civil", "civillight", "meteo" or "two", and output (xml or json)
